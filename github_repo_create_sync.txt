
common way to create a repo in github:
#1 create new remote repo from github then clone to local: 
(SSH key already set in local and pub key add to github account, so okay to use SSH way)
1. create new repo on github using "+", then add new repository
2. open a local terminal, then under dir where new repo locate (workspace): git clone git@github.com:chevernemilia/test_repo_creation.git
3. better create a master branch by: git checkout -b master and commit back to remote (work on master than main)
